This starter is meant to be changed. Begin with small requests:

1. Replace the site name and contact email.
2. Rewrite the homepage for your audience.
3. Replace the sample images.
4. Add one page.
5. Run the build before deploying.

The useful habit is not asking the agent to do everything at once. Ask for one change, verify it, then move to the next.
